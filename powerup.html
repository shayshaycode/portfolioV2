PowerUp Review Game

PowerUp is a fully custom-built, Jeopardy-style review game that empowers teachers to create interactive content for their students. Designed using full-stack development principles, it features a sleek interface, question board structure, and an integrated database for storing and managing custom games.

Built from the ground up using HTML, CSS, JavaScript, and backend technologies, PowerUp promotes in-class engagement while offering a flexible platform for formative assessment. While currently optimised for in-person use, its modular build leaves room for future enhancements like online multiplayer functionality.
